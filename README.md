# NTE Auto-Fish

Автоматизация рыбалки в Neverness To Everness (NTE) с GUI на CustomTkinter.

![Иконка](icon.png)

## Дисклеймер

Использование автоматизации может нарушать пользовательское соглашение игры
и привести к бану аккаунта. Использование на свой страх и риск. Автор не несёт
ответственности за последствия. Не используй на основном/ценном аккаунте.

## Что умеет

- **Авто-рыбалка** — заброс, подсечка, мини-игра QTE (детект полосы и удержание A/D)
- **Авто-продажа рыбы** через меню Q
- **Авто-закупка наживки** через меню R, с поиском нужной плитки через template matching
  (не зависит от позиции наживки в магазине)
- **Счётчик забросов** вместо детекта баннера "нет наживки" — детерминированно и без ложных срабатываний
- **Антибот-маскировка** — случайные паузы, jitter по координатам и таймингам, длинные перерывы
- **Visual picker** — кликаешь "Указать" → переключаешься на игру → пробел → выделяешь нужное место → готово
- **Пресеты разрешений** — 1080p / 2K в один клик
- **Статистика** — рыб/час, успешность, средние QTE и реакция, за сессию и за всё время
- **Иконка приложения** + плавающий status-оверлей поверх игры

## Установка (для пользователя)

1. Скачай последний [Release](../../releases) — архив `NTE_Fish.zip`
2. Распакуй куда удобно
3. Запусти `NTE_Fish.exe` — Windows попросит разрешение (UAC), это нужно для имитации нажатий
4. Игра должна быть в `Borderless / Windowed Fullscreen` 1920×1080 (на 2К нужен пресет — см. ниже)

## Первый запуск

1. **Шаблон наживки.** Положи в папку `templates/bait_target.png` скриншот мешочка наживки
   из меню магазина. Win+Shift+S → выдели только сам мешочек (без подсветки бордюра
   и без названия снизу) → сохрани как `templates/bait_target.png`. README в этой папке
   тоже подскажет.

2. **Координаты.** Если играешь на 1920×1080 — дефолты должны подойти. Если нет:
   - Открой вкладку **Координаты**
   - Выбери в выпадашке готовый набор (например 2560×1440) → нажми "Применить" → "Сохранить"
   - Либо откалибруй каждую координату вручную через кнопку "Указать"

3. **Настройки наживки** на вкладке **Главная**:
   - "Забросов до первой авто-закупки" — сколько у тебя сейчас наживки на руках
   - "Сколько закупать за раз" — сколько брать в магазине (1..99)

4. **Запуск.** Заходишь в игру, встаёшь у удочки, жмёшь хоткей старт/стоп.

## Хоткеи (по умолчанию)

| Клавиша | Действие |
|---------|----------|
| F9 | старт/стоп |
| F12 | выход |
| F7 | разовая продажа |
| F6 | разовая закупка наживки |
| F3 | вкл/выкл авто-продажу |
| F4 | вкл/выкл авто-закупку |

Все переназначаются на вкладке **Хоткеи**. F-клавиши выбраны чтобы не конфликтовать
с вводом цифр в полях GUI и чата игры.

## Troubleshooting

**SmartScreen ругается** — false-positive из-за PyInstaller. "Подробнее" → "Выполнить в любом случае".

**Не находит плитку наживки** — пересохрани `bait_target.png` чище (только мешочек, без
бликов и текста). Либо понизь `BAIT_MATCH_THRESHOLD` на вкладке Координаты до 0.6.

**Клики промахиваются на 2К/4К** — применил пресет, но не калибрует точно?
Пройдись через "Указать" по конкретным кнопкам которые мажут.

**Бот стопится с "автопокупка: ошибка"** — проверь `app.log` рядом с .exe.
Чаще всего: устарели координаты после обновления игры.

## Разработка / сборка из исходников

```bash
# venv + зависимости
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

# Запуск из исходников
start.bat

# Сборка .exe
build.bat
# Результат → dist/NTE_Fish/NTE_Fish.exe
```

## Структура проекта

```
nte_fish/
├── main.py              # entry point
├── gui.py               # CustomTkinter UI
├── bot.py               # FSM рыбалки + авто-цикл
├── controller.py        # SendInput для DirectX игр
├── detectors.py         # HSV для QTE
├── shop_detect.py       # template matching мешочка
├── picker.py            # визуальный выбор координат
├── humanizer.py         # антибот jitter
├── overlay.py           # плавающий статус
├── stats.py             # запись stats.jsonl
├── settings.py          # user_settings.json
├── presets.py           # 1080p / 2К
├── paths.py             # пути для frozen vs script
├── config.py            # дефолты
├── make_icon.py         # генератор icon.ico
├── nte_fish.spec        # PyInstaller spec
├── build.bat            # сборка
├── start.bat            # запуск
└── templates/
    ├── bait_target.png  # твой шаблон (не в репо)
    └── examples/        # картинки для picker (опц.)
```
